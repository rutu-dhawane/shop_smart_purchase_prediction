# Shop Smart Purchase Prediction

This project predicts customer purchase behavior using a Decision Tree Classifier and an end-to-end machine learning pipeline.

## Key Achievements

- Built a customer purchase prediction model using Decision Tree Classification.
- Implemented data preprocessing using ColumnTransformer and OneHotEncoder.
- Created a Scikit-learn Pipeline for automated preprocessing and training.
- Optimized model performance using GridSearchCV with 5-fold Cross Validation.
- Visualized the final Decision Tree for model interpretability.

## Model Performance

- Test Accuracy: 84.39%
- Best Cross-Validation F1 Score: 0.6344

## Technologies

Python, Pandas, Scikit-learn, Matplotlib

## Outcome

The optimized Decision Tree model successfully predicts customer purchase behavior and demonstrates a complete machine learning workflow, including preprocessing, hyperparameter tuning, evaluation, and visualization.
